# Sample images for the demo

Drop 5-10 PlantVillage test images here for the demo (the .gitignore lets
this directory be tracked, but the .gitignore also excludes large image
binaries if you add ignore patterns later).

## Where to get them

- Repo: https://github.com/spMohanty/PlantVillage-Dataset
- Or Kaggle: search "plantvillage dataset"

Good demo picks (clear visible disease):

- Tomato early blight
- Tomato late blight
- Tomato bacterial spot
- Apple scab
- Potato early blight

Save as `<crop>_<disease>.jpg` for clarity, e.g. `tomato_early_blight_01.jpg`.
